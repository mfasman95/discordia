Usage

Use this generator to quickly bootstrap the sheer minimum discordia action that can be shared and used in other bots created with the @discordia/framework. Once this generator is complete you are a single npm publish away from a sharable action.

# Run the module using npx
npx @discordia/create-discordia-action

# Run the module using yarn
yarn create @discordia/discordia-action
